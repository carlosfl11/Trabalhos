3D Studio Max
Tire tutorial

This tutorial will show you how to make a simple tire in Max 5. We will be making a truck tire, but this tutorial can be applied to any other type of tire you wish to build.

1) Find a tread pattern you wish to use and in max using the top view create simple polygon's that look similar to the tread you are trying to copy.

2) Once you have created the tread pattern you need to connect the treads together and make a section of the side wall. This tread section will need to be made so it will make a repeating pattern.

3) Now we need to give the tread some depth. The faces that are already selected in this image are the ones we will extrude. Extrude these faces to a height your happy with then extrude them one more time, but only a small amount. Select the scale tool and scale the top faces down to about 98%. This will give you a slight edge on the tread.

4) In our front view we need to weld the vertices of our outside treads to the side wall. Select both sides and pull the vertices down until they line up with the side wall vertices and weld them together.

5) We need to select the tire tread and we are going to duplicate it. In the right view click on the X axis hold down the shift key and drag the tread to the left. A Clone Options box will appear select copy and under "Number of Copies" type in 30. Make sure you line up the edges of the tread pattern when duplicating it. Now we need to attach them all so its one object. Select the original pattern we copied and click on the "Attach" button and start attaching the duplicated patterns. Now select all the vertices because we need to weld the ones together were each copied section sits together.

6) Now we will apply two bend modifiers to this tire tread. The Angle on the first bend modifier needs to be set to "5.0" and the "Bend Axis" should be set on "X". The Angle of the second bend modifier needs to be set to "-367.5" the "Direction" should be set to "90.0" and the "Bend Axis" should be set to "Y".

7) Well that's it. Form this point you can make any type of edits you wish to get the desired effect.
