BUICK WILDCAT 1966 TUTORIAL

Model

The car is a 66 Buick Wildcat modelled in 3D Studio Max 6.0, integrated in a real photo and rendered with VRAY using as reference a model kit.

Texturing

The car´s texture has been used mainly to add dirt and dust to the model using some dirty masks included in the TOTAL TEXTURES Nº8

Wheel

The procces has been the same for all the model components, headlights, wheels,... I make a viewport capture in Photoshop and I use this graphics/tut_calero_image as base for painting the texture with the dirty maps from the CD Nº8. We have to adjust the dirt over the surface using the wire as a base. After painting the texture I create a Blend material using two basics materials. One of this material is the base material (wheel, metal, chrome...) and the second one is a brown mate material that simulate the dust and dirt. For mixing this two material I use the mask that I have painted before.
