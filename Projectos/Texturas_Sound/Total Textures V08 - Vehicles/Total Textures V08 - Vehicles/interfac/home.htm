Welcome to Total Textures Vol 8 'Vehicles'.

Total Textures v8 'Vehicles' is a collection of textures for texturing Car Models and scenes. The texures range from Tyre bump maps to cool flame decals. In the case of textures such as decals, we have included the alpha-mapped TIFF files. Pre Alpha-mapped TIFF files give transparency to the outer edges of a map. For The first time on a Total Texture CD we have included .dxf meshes of some of the more 'common' car objects. These include Alloy Wheels, brake calipers, dials etc. These mesh downloads (where available) are featured alongside the Tiff downloads on each page, which can be dowloaded by clicking the icons shown below when they appear in between the colour and bump map thumbnails of a texture. (in the case of meshes the displayed picture is a rendering of a post-textured version of the object).

Extras Icons:-

There are Over 300 Tiff files (with Alpha Channel), 45 DXF Meshes, 45 .3ds meshes, 13 paintshader files for lightwave, 13 paintshader files for Cinema 4D and 10 Paintshader Files for 3DStudio Max included on this CD. If a Texture Map or reference image has any of these file,s the following icons will appear between the colour ad bump/opacity map in the browser window.
Tiff with Alpha Map download
DXF Mesh download
3DS Mesh download
Paint Shader download

To Save these extra features you can either:

PC Users : Right Click the Icon and choose 'Save Target as' from the drop down menu, the browse to the required folder.

Mac Users : Hold 'Option Key' and click the Icon. Choose 'Save Target as' from the drop down menu then browse to the required folder.

For TIFF's Only : Quicktime Pro Users (Mac & PC) : Click the Icon and the Image will load in a new window, then right (option) click the image for the quicktime save options menu.
