Welcome to this tutorial about texturing and rendering of a car. The softwares that I have used are 3ds MAX 5.1, Brazil rendering system and one small but very useful plug-in named "Texporter". The car is Mercedes AMG, which I modeled a year back. The modeling was done in Rhinoceros (NURBS modeler) v 2.0. Am in the process of writing a tutorial for it but for now, texturing it is all that’s I am going to discuss. So lets move ahead with it.

BONNET:

I will start with the bonnet of the car. We will be giving the bonnet a black color with the flames map on it.

  1. First I select the bonnet, right click and select " Hide Unselected ". This hides all the objects other than the bonnet and helps in making quick test renders.
  2. Then add the "UVW map" modifier for the modifier list. Select "Bitmap Fit" and select the image file ( the yellow image in this case ) from the list. The gizmo takes the size of the bitmap specified. To adjust the size of the gizmo, select " gizmo" from the subject level of the UVW map and uniformly scale the gizmo till it perfectly fits the surface using the scale tool.
  3. Then open the material editor, select an empty slot and change the default "standard" material to "blend" material.
  4. In material 1 of the blend material, we will be assigning a black reflective material to the bonnet. Set the parameters as shown in the image.
  5. In material 2 of the blend material, we will assign another material to get the flames on the bonnet. In the diffuse map slot, select the map that is to be applied and uncheck the "tile" boxes to prevent the map from tiling and appearing more than once.
  6. In the mask slot of the "blend" material, we will use the alpha of the flame graphics/tut_thorat_image. As u know, alpha is an image with whites , blacks and in-between grays. The areas where the image is black, material 1( black body paint) will be visible and the areas where the image is white, material 2 ( flame image ) will be visible. Thus the resulting render will be something like this. Do not forget to uncheck the tiling options of this map.

Material 1 and material 2 of the blend material are independent material with their independent properties. Both of them can be controlled from their respective slots.

Most of the mapping is done in a similar fashion. Sometimes, when the material map is not visible in the viewport, it becomes a pain adjusting the UVW gizmo for perfect placement and alignment of the map. I’ll explain how the dirt map on the car was done using Texporter.

Texporter is a plugin, which unwraps the UVW map and saves it as an image file, which can be used as a reference to paint textures by opening the file in mage editing software like Photoshop etc.

 

 

SIDE OF THE CAR WITH DIRT:

I select the surface and apply it a UVW map modifier with planar projection.

In the utilities panel, click on "Texporter". The image size determines the size of the image that will be generated. The values of width and height should be in the same ratio as that of the UVW map gizmo. Click the "pick object" tab and select the surface to be mapped. It generates an image file, which could be saved as jpeg and opened in Photoshop.



image generated by Texporter



Dirt map created in photoshop

Alpha map of the dirt map.

Keeping this image on the bottom most layer as a reference, the dirt map is painted on other layers. This dirt map, along with its alpha map, can then be applied to the surface in a similar way in which the flame image was applied to the bonnet.

Also, a bump map can be applied to give the dirt a little raised feeling.

Headlights:

I wanted to get beautiful headlights, and this surely requires an effort. I could not find a map, which would do the work coz it was looking very unreal. So I actually modeled the interior of the headlights and applied it a fully reflective material, almost mirror like.

The outer cover of the headlight has almost transparent material with a "chequered flag" like map in the bump slot to simulate the various refractions of the glass.

There is an HDRI map in the environment slot with spherical coordinates, which is giving beautiful reflections in the headlight.

The lighting setup is simple, with only the skylight on, and the ground plane enabled. Other than skylight, no other lights are used in the scene.

I hope this would help you in getting the desired result.
